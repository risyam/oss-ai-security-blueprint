# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do not disclose it publicly.
Since this is an educational security blueprint, some vulnerabilities are intentional for demonstration purposes (e.g. inside `Vulnerable-Version` folders). 

However, if you find unintended vulnerabilities in the `Secured-Version` apps, the `secure-lib`, or the infrastructure code, please report them.

### To Report:
Please open an issue in the repository privately (if the platform supports it) or email the maintainers.
