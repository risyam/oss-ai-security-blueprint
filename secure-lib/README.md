# secure-lib

Reusable AI/LLM security components for the OSS AI Security Blueprint.

## Modules

- `scanners`: prompt injection, secrets, and prompt-leak detection
- `retrieval`: RAG context sanitization and document provenance validation
- `guardrails`: output validation and schema enforcement
- `policy`: instruction hierarchy and tool permission management
- `monitoring`: rate limiting and token usage enforcement

## Install

```bash
pip install -e secure-lib
```
